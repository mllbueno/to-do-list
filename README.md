# :sparkles: My To Do List :sparkles:

## Como executar

```sh
npm i
npm start
```

## Como testar

```sh
npm test
```

## Estrutura do Projeto

```
├── public
├── src
│   └──__test__      // Arquivos de teste
│   └── assets       //images
│   └── components   // Components usados para criação das páginas
│       ├── Form
│       ├── Todo
│       ├── TodoList
│   └── pages        // Páginas da aplicação
│       ├── App
│   └── styles
│       ├── App.css
│   └── Index.jsx
```
